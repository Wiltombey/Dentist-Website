# Dentique – Dental Clinic Website

A responsive dental clinic website built using HTML, CSS, and JavaScript. The project focuses on clean design, smooth interactions,
and a realistic business layout.

## Features

* Responsive design for mobile and desktop
* Smooth scrolling navigation
* Mobile hamburger menu
* Scroll-based animations
* Button click effects
* Hero section with parallax effect
* Testimonials slider with auto and manual controls
* Contact form with basic validation

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)

This is a frontend-only project. The contact form is not connected to a backend.

## Future Improvements

* Connect form to a backend (Node.js, Firebase, etc.)
* Add appointment booking functionality
* Improve accessibility
* Optimize performance

## Purpose

This project was built to practice frontend development and create a portfolio-ready website based on a real-world use case.
